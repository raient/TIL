#  마크다운 문법 정리



## 1. 제목(heading)

- 제목은 # 의 개수로 표현된다.
- H6, 제목을 6단계까지 구분하여 작성할 수 있다.
- 제목을 잘 작성해두면, Overview나 목차로써 활용 가능하다.

# 제목1

## 제목2

### 제목3

#### 제목4

##### 제목5

###### 제목6



## 2. 목록

- 순서가 없는 목록, *, - 으로 작성
- 순서가 있는 목록, 1. 으로 작성
- 탭 및 시프트탭으로 계층의 이동이 가능



* 순서가 없는 목록
  * 계층이 존재

1. 순서가 있는 목록

   1.1 순서가 있는 목록
